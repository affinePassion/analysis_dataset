# ⚽ UEFA Champions League 2016-2022 Analytics Dashboard

## 📖 Описание проекта

Веб-приложение для анализа данных Лиги Чемпионов УЕФА за период 2016-2022 годов. Проект разработан в рамках практической работы №2 по дисциплине "Анализ больших данных".

Приложение предоставляет:
- Интерактивную визуализацию схемы базы данных
- Исследовательский анализ данных (EDA) с графиками и таблицами
- Модель машинного обучения для прогнозирования посещаемости матчей

## 🚀 Функциональность

### 📖 Описание проекта
- Общая информация о проекте и источнике данных
- Статистика датасета (количество записей в каждой таблице)
- Интерактивная схема базы данных с отображением связей между таблицами
- Описание первичных и внешних ключей

### 📊 EDA анализ
- **Топ-10 команд** по посещаемости домашних матчей
- **Топ бомбардиров** турнира за весь период
- **Статистика по сезонам** (голы, посещаемость, тренды)
- **Анализ стадионов** (количество матчей, вместимость, география)

### 🤖 ML модель
- **Прогнозирование посещаемости** матчей Лиги Чемпионов
- Модель: `RandomForestRegressor`
- Метрики качества: MAE, RMSE, R² Score
- Интерактивное предсказание с выбором параметров матча
- Визуализация прогноза в виде gauge-диаграммы

## 📦 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/affinePassion/analysis_dataset
cd practics/practice_2
```

### 2. Создание виртуального окружения
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Запуск приложения
```bash
streamlit run app.py
```

Приложение откроется в браузере по адресу: http://localhost:8501
